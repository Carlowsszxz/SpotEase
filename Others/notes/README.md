## Application HTML Pages Overview

This section documents the purpose and access level for each major HTML frame/page in the system. Use this guide for AI reasoning, onboarding, or context extraction.

---

### Admin HTMLs (Admin-Only Pages)

These pages are accessible **only to system administrators**:

- **FrameAdminPanel.html**  
  *Full administrative dashboard: manage resources, sensors, users, and system settings.*

- **FrameAnalytics.html**  
  *View analytics and system reports: usage statistics, occupancy trends, performance metrics.*

- **FrameAuditLogs.html**  
  *Access audit logs: track system actions, security monitoring, and event histories.*

- **FrameSensorReadings.html**  
  *Inspect raw sensor/IoT data: for diagnostics, troubleshooting hardware, and real-time monitoring.*

---

### User HTMLs (Regular Users: Students, Faculty)

These pages are accessible to **all authenticated users**, including students and faculty:

- **FrameDashboard.html**  
  *Main user dashboard shown after login; quick access to summary, notifications, and actions.*

- **FrameMap.html**  
  *Visual map/grid of resource availability, e.g. parking spots, seats, rooms.*

- **FrameMyReservations.html**  
  *Manage your reservations: view, create, or cancel resource bookings.*

- **FrameProfile.html**  
  *Your account page: edit personal info and view account details.*

- **FrameAccessHistory.html**  
  *User-facing RFID access history: view recent tag tap events and timestamps.*

- **FrameRegister.html**  
  *New user registration page.*

- **FrameLogin.html**  
  *Login page for all users (regular and admin).*

- **FrameReservation.html**  
  *Reserve a spot, seat, room, or other resource.*

- **FrameResourceList.html**  
  *Browse or search all available resources.*
